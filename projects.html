Proyectos mios o en los que he participado que tienen cierta importancia

Podreis ver todo tipo de proyectos separados por categorias de los cuales podreis probar si ningún problema, siempre y cuando tengáis todo lo necesario

Audio

Chromecaster GUI

Javascript C++ – Electron (Chromium + Nodejs)

Sencilla aplicación que permite coger el sonido que llega al ordenador (dispositivos de entrada) y enviarlo a un Chromecast en directo usando MP3 o FLAC. Soporta Windows, Linux y macOS.

butt (Broadcast Using This Tool)

C C++ - FLTK

Herramienta multiplataforma para streaming the audio por internet. Envia contenido a servidores Icecast y Shoutcast, usando MP3, Vorbis, Opus o AAC, además permite grabar. Además de multi-idioma. Es un fork.

youtubeAudio

JavaScript (React) + Backend en Python

Web que permite escuchar videos de Youtube. Desarrollada en conjunto con MrRaulWhite, alkesst, amgxv y yo, usa youtubedl-audio-api, una API hecha en Python y Flask para obtener las URLs de los videos usando youtube-dl.

Utilidades

LoL Item Sets (Mac)

Swift

Añade sets personalizados generados automaticamente dependiendo del win rate y el más usado para cada campeón de League of Legends. Este cliente es solo para macOS, pero existe una versión para Windows. Visitad la web para mas información

DADiskActivity

Objective-C

Icono en la barra superior de macOS que muestra la actividad de un dispositivo de almacenamiento como el disco duro de tu Mac.

WikiMusic

Java (Android)

Proyecto para la Universidad. Aplicación que busca información sobre un artista, un álbum o una canción usando la API de Last.FM. Requiere Android 4.4 o superior. Descargar APK .

watch-time

Objective-C++ / C++

Muestra la hora, pero de forma minimalista. Para macOS (10.6 o superior) y Linux (GTK+ 3).

iTunes Scrobbler

Swift

App para macOS que permite hacer scrobblings a Last.fm desde iTunes sin tener que aguantar al cliente oficial de lo mal que va. El programa reside en la barra superior de macOS y desde ahi puedes cambiar los ajustes. Requiere macOS 10.12 o superior.

Widgets

LastFM Web Widget

Javascript - jQuery

Curioso widget que muestra tus últimos scrobblings. Tema personalizable.

Playbox.widget

CoffeeScript + Objective-C

Widget para Übersicht que muestra una carátula, el progreso y los metadatos de la canción que estés escuchando en iTunes, Spotify o VOX.

Bibliotecas

node-flac-bindings

node.js (Javascript + C++)

Utilidad

Te permite llamar a las funciones de la API de flac directamente desde nodejs. También incluye clases de utilidad para codificar y decodificar rápidamente usando clases y flujos de node.

async-net

Java

Networking

Crea sockets TCP o UDP en Java y úsalos de forma asíncrona, aprovechándose de las lambda funciones de Java. Diversas opciones y configuraciones para tener sockets funcionando. Perfecto para interfaces de usuario. Funciona con Java 7 o superior y con Android 4.2 o superior.
